1. Se debe tener varios servidores de archivos (Administrar jerarquía de directorios)
	Es decir por cada cliente sabe donde están los archivos como son varios entre si deben tener el mismo dato actualizado
2. Cada servidor va a manejar un árbol donde estan los archivos del sistema
	Es decir en la forma de mostrar es como un arbol de carpetas (los nombre de carpeta pueden ser autogenerado
3. Se puede escoger, en relación con el acceso, el montaje de archivos o el acceso remoto.
4. Se debe manejar la transparencia de los nombres
5. Cada archivo debe tener al menos 2 copias
	las copias deben estar en clientes (en el caso cuando son los primeros conectando deben esperar a que esten los otros 2 clientes)
6. Los servidores deben manejar archivos log de los procesos
	
7. Para la actualización de las copias, se debe implementar uno de los 4 algoritmos
	Por ejemplo puede actualizar el base y luego los demas
